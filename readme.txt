== Description ==

The easiest way to include Google Analytics, Clicky Analytics, ClickCease and CallRail into your Wordpress site.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/includer` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Includer screen to configure the plugin
